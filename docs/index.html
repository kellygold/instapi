Why InstaPi

Privacy first

Runs entirely on your Pi. No cloud servers, no tracking, no data collection. Your photos stay yours.

No subscriptions

One Raspberry Pi. That's the whole cost. No monthly fees, no premium tiers, no in-app purchases.

Open source

MIT licensed. Fork it, modify it, make it yours. The code is all on GitHub.

Self-healing

A watchdog monitors everything and auto-recovers from crashes, network drops, and USB issues. Set it and forget it.

How it works

1

Install on your Pi

One command. Everything sets up automatically.

curl -sSL https://raw.githubusercontent.com/kellygold/instapi/main/pi-setup/install.sh | bash
2

Scan the QR code

The frame shows a QR code on first boot. Scan it with your phone.

3

Add photos

Upload from your phone, pick from Google Photos, or share the link with family. Photos appear on the frame instantly.

For makers

Built on a Raspberry Pi. Runs Flask. Talks to your frame.

Two display modes

  • USB mode — Pi appears as a USB drive. Works with any photo frame that has a USB port.
  • HDMI mode — Web-based slideshow via Chromium kiosk. Fade, slide, zoom transitions + Ken Burns.

Tech stack

  • Python 3 + Flask
  • Google Photos Picker API
  • Pillow for thumbnails and watermarks
  • systemd services + watchdog cron
  • ngrok for remote access

Admin panel

Manage photos, slideshow settings, system info, and more — all from your phone. Dark theme, mobile-friendly, built-in help guide.
